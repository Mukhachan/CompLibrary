ТЗ для библиотеки.

Концепт:

Человек приходит в библиотеку, прикладывает карту, выбирает книгу уходит. 
В этот момент в базе данных отмечается кто и когда забрал книгу.

Аналогично когда человек возвращает книгу.

Также должна быть система ролей:
- Класс. Рук.
- Ученик
- Библиотекарь
- Администратор

ЯП:
- Python (Flask)
- HTML
- CSS
- SQlite
- JavaScript (Возможно в будущем)


Возможности ролей (по возрастанию): 
    Ученик: (Не приступал)
        - Просматривать список доступных книг
        - Брать книгу
        - Регистрироваться/авторизироваться

    Классный руководитель: (Не приступал)
        - Просматривать список доступных книг
        - Брать книгу
        - Регистрироваться/авторизироваться
        - Просматривать список должников своего класса

    Библиотекарь: (В разработке)
        - Имеет полный доступ к б/д с должниками и людьми взявшими книги
        - Всё что могут люди ниже ролью


Столбцы БД:
- id
- btitle
- author
- year  (не ищется)
- number (не ищется)
- descript
- dt_string
